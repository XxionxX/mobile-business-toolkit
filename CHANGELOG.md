## [0.1.0] - 2025-02-XX
### Added
- Initial repository structure and modular architecture
- Payroll calculator with:
  - currency formatting
  - percentage formatting
  - zero-division safeguards
  - warning messages
  - persistent input storage
  - copy-to-clipboard functionality
- UI layout and reusable input components
- Local development instructions
- Private GitHub repository setup
- Apache 2.0 license
- Project README
- Improved payroll UI styles (modernized inputs, padding, rounding)
- Added main menu button to payroll screen
